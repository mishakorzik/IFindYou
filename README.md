### Installing

* `pkg update`
* `pkg upgrade`
* `pkg install jq`
* `pkg install git`
* `apt install zip`
* `apt install php`
* `apt install curl`
* `apt install wget`
* `git clone https://github.com/mishakorzik/IFindYou`
* `cd IFindYou`
* `bash install.sh`
* `ngrok authtoken <token>`

#### Finished you have successfully downloaded IFindYou now to launch!

* `cd IFindYou`
* `bash main.sh`
